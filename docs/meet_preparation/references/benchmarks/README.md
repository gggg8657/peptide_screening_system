## 1. 혈청 반감기 예측 도구 비교

| 도구 | 예측 환경 | 방법론 | 훈련 데이터 | 성능 지표 | 펩타이드 유형 | 권장 용도 |
|------|---------|--------|-----------|---------|------------|---------|
| ProtParam (ExPASy) | 세포내 (CHO/E.coli) | 경험적 규칙 (N-end rule 기반) | 표준 N-end rule 테이블 | 정성적 (안정/불안정) | 선형 단백질 | 1차 스크리닝 |
| PlifePred | 포유류 혈중 | ML (PaDEL descriptors) | 163 자연 + 98 변형 펩타이드 | r=0.692 | 자연+변형 모두 | 혈중 반감기 정량 예측 |
| HLP | 장내 프로테아제 환경 | ML | 장내 안정성 실험값 | — | 자연+변형 | 경구 투여 검토용 |
| PeptideRanker | — | N-to-1 신경망 | 다양한 bioactive peptide DB | 확률값 0~1 | 20aa 이하 | bioactivity 스크리닝 (반감기 아님) |

**SST14 analogue 권고**: PlifePred (혈중 반감기 직접 예측) + ProtParam (보조 확인)

## 3. 디퓨전 도킹 / 구조 예측 비교

| 도구 | 방법론 | 입력 | 친화도 예측 | 펩타이드 도킹 | 성능 벤치마크 | 라이선스 |
|------|--------|------|-----------|------------|------------|---------|
| DiffDock | 디퓨전 생성 모델 | 단백질 + 리간드 SMILES | X (포즈만) | 부분 (소분자 특화) | PDBBind top-1 38% (RMSD<2Å) | MIT |
| Boltz-2 | 트랜스포머 + diffusion | 서열 / 구조 | O (binding affinity) | O (펩타이드 포함) | CASP16 affinity 1위; FEP 대비 1000배 빠름 | MIT |
| AlphaFold3 | transformer + diffusion | 서열 | X | O (단백질-리간드 복합체) | CASP15 구조 정확도 1위 | 비상업 제한 |
