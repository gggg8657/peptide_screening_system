플랫폼 소개

무엇을 하는 시스템인가?

SSTR2 AI Co-Scientist는 자율형 에이전트 기반 신약 후보 발굴 플랫폼입니다. 모체 소마토스타틴 펩타이드 SST-14 (AGCKNFFWKTFTSC, Cys3–Cys14 이황화결합을 가진 14잔기 환형 펩타이드)의 변이체를 스크리닝하여 SSTR2 (소마토스타틴 수용체 2형)에 더 강하게 결합하면서, 혈중 반감기가 길고 근연 off-target(SSTR1/3/4/5)에는 선택적인 후보물질을 탐색합니다.


Planner · Critic · Reporter 3종의 로컬 LLM 에이전트가 폐쇄형 자기개선 루프를 구동하며, PyRosetta FlexPepDock을 통한 실제 전원자 분자역학 도킹으로 결합 에너지 ΔG를 산출합니다. 모든 과정은 실시간 웹 대시보드를 통해 투명하게 모니터링됩니다.


주요 대상 연구자: 방사성의약품 · 펩타이드 치료제 연구팀 (SSTR2-targeted PRRT / DOTATATE 계열 연구), 조기 인-실리코 스크리닝 주기 단축을 원하는 팀, 과학적 엄밀성과 감사 추적성을 포기하지 않는 연구자.

🔬

🧠
로컬 LLM — 외부 API 의존 없음 Qwen3-32B를 vLLM으로 로컬 서빙 (포트 8000, OpenAI 호환). 데이터 유출 없음.
🎛️

🛡️
반(反)환각 가드 레이어 pharmacology_guards.py가 문헌값·유효 범위·부호 규약을 강제하고 신뢰도 등급을 태그.

핵심 기능

11가지 핵심 기능

자율 루프부터 물리 도킹, Pareto 최적화, 반(反)환각 가드까지 — 전체 스크리닝 파이프라인을 하나의 시스템으로.

01 🔄

Agentic AI Discovery Loop

Planner · Critic · Reporter — 지치지 않는 AI 연구자가 루프를 구동합니다.

3개 LLM 에이전트가 N-이터레이션 루프를 주도. Planner가 목표·목적 모드(auto / ddg_only / ddg_plus_constraints)를 설정하고, Scientist Critic은 실패를 진단해 라운드당 최대 2개의 파라미터 변경을 제안(adaptive gating), Reporter는 이터레이션 노트북·랭킹·PyMOL 렌더 스크립트를 작성합니다. 모든 에이전트는 Qwen3-32B (vLLM)으로 로컬 서빙 — 외부 API 의존 없음.
02 ⚗️

Physics-Based Docking (PyRosetta FlexPepDock ΔG)

결합 점수는 추측이 아닌 진짜 전원자 분자역학에서 나옵니다.

각 후보 복합체를 PyRosetta FlexPepDock으로 정제 후 InterfaceAnalyzerMover로 계면 결합 에너지 ΔG 계산 (ΔG = E(complex) − E(receptor) − E(peptide), kcal/mol / Rosetta Energy Units). 입체 충돌 점수(fa_rep)도 함께 보고. 게이트: ΔG ≤ −5.0 kcal/mol, 충돌 ≤ 10 REU.
03 🎯

Multi-Objective Screening

ΔG · 반감기 · 선택성 · ADMET — 4가지 목표를 동시에 최적화합니다.

비용 계층형 목적 스택(multiobjective.py)이 결합 ΔG, 서열 기반 반감기 추정, SSTR2 선택성 마진, ADMET 합리성 대리지표(Instability Index · GRAVY · Boman index · pI)를 결합. 기본 목적 가중치: ΔG 40% · 선택성 25% · 안정성/반감기 20% · ADMET 15%.
04 🔒

SSTR2 Selectivity Off-Target Screening

SSTR1/3/4/5 4형제 수용체 대상 실제 도킹으로 선택성을 증명합니다.

상위 후보를 큐레이션된 단일 체인 SSTR1, SSTR3, SSTR4, SSTR5 수용체 구조(AlphaFold 유래, SSTR2 프레임 사전 정렬)에 동일한 FlexPepDock + InterfaceAnalyzer 파이프라인으로 도킹. selectivity_margin = ΔG(worst off-target) − ΔG(SSTR2); 양수 = SSTR2 특이적. 게이트: 마진 ≥ 10 kcal/mol.
05 📈

Bayesian Optimization + Thompson Sampling

어디를 변이시킬지 학습하여 실험 예산을 더 스마트하게 씁니다.

Thompson-sampling 다중-무장 밴딧이 변이 가능 위치별 Beta(α, β) 신념을 유지하고 매 라운드 집중 위치를 샘플링(bandit.py). 보완적인 GP 베이지안 최적화기(BoTorch qNEHVI 다목적 획득 함수, ESM-2 또는 one-hot 임베딩)가 다음 변이를 제안(bayesian_optimizer.py).
06 📊

Multi-Trial Statistical Validation

단일 도킹 노이즈를 제거 — 여러 시드로 재현성을 검증합니다.

설정 가능한 N-trial 검증 (1=off · 3=빠름 · 5=표준 · 10=논문 수준)이 최종 후보를 여러 시드로 재도킹하여 ΔG의 중앙값 / 평균 / 표준편차를 보고하며 CV(변동계수)가 낮으면 조기 중단. 측정 1회 서열은 UI에서 "재현성 미검증"으로 표시.
07 🔗

Disulfide-Bond Preservation (Cys3–Cys14)

SST-14를 약물로 만드는 환형 골격이 항상 보호됩니다.

변이 엔진이 Cys3 · Cys14 앵커 위치를 변이 금지 구간으로 처리. 주변 위치만 변이하여 Cys3–Cys14 이황화결합을 보전. UI에서 "Disulfide Constraint" 토글로 노출.
08 🧩

Pharmacophore (FWKT) Constraints

SSTR2 인식 핵심 구간을 고정하여 변이체가 수용체 결합 능력을 유지합니다.

FWKT 약효단 (7~10번 위치)은 불변으로 고정. 제안된 모든 변이체는 _preserves_pharmacophore()로 검사되며 위반 시 최대 3회 재시도 후 폴백 (PHARMACOPHORE_RETRY_LIMIT = 3).
09 📐

Pareto Ranking (NSGA-II)

가중합 편향 없이 진정한 트레이드오프 경계면을 도출합니다.

NSGA-II 비지배 정렬 + 군집 거리(pymoo)로 {ΔG, 안정성, 약물 가능성, 다양성} 목적 집합에 대해 후보를 랭킹 (pareto_ranking.py). 충돌 및 하드-위반 제약 조건 하에 기존 가중합 점수를 대체.
10 🖥️

Web Dashboard — 실시간 Mission Control

AI 연구자의 작업을 실시간으로 감시하고, 3D 구조를 검사하며, 클릭 하나로 실험을 제어합니다.

React + Vite 대시보드 ↔ FastAPI 백엔드. 실시간 에이전트 타임라인 · 루프 진행도 · 수렴 그래프 · ΔG 분포. Mol* (molstar 5.6.1) 3D 뷰어 (복합체 / 카툰 / 볼앤스틱 / 표면 4종 모드) + PyMOL 렌더 갤러리. 정렬 가능한 후보 테이블 (ΔG · Total Score · Clash · Final Score · 재현성 · 도구설명). 실험 파라미터 설정 및 Start / Stop 제어.
11 🛡️

Pharmacology Guards — Anti-Hallucination

AI가 날조한 약리학 수치가 결과에 포함되는 것을 전용 가드 레이어가 차단합니다.

독립적인 선언형 가드 모듈(pharmacology_guards.py)이 문헌 근거 값·유효 수치 범위·부호 규약을 강제하고, 휴리스틱 함수를 정직하게 레이블링하며, API 응답에 신뢰도 등급을 태그. 회귀 테스트 39+개로 백업.

과학적 검증

정직하게 공개합니다 — Scientific Validity Note

AI 플랫폼이 무엇을 실측하고, 무엇을 추정하는지 명확히 구분합니다. 이 투명성이 플랫폼의 핵심 차별점입니다.

실제 물리 측정값

결합 ΔG — 진짜 전원자 인터페이스 에너지

PyRosetta InterfaceAnalyzerMover가 FlexPepDock 정제 복합체에서 계산 (kcal/mol / Rosetta Energy Units). 실제 전원자 계면 에너지이지만 절대 친화도(Ki/Kd)가 아닌 상대 점수임을 명시합니다.

실제 물리 측정값

선택성 마진 — 실제 off-target 도킹 결과

SSTR1/3/4/5 대상 실제 도킹에서 나온 수치입니다. 동일한 FlexPepDock + InterfaceAnalyzer 엔진으로 계산되며, 양수 마진(≥ 10 kcal/mol)은 진짜 수용체 아형 분별 능력을 의미합니다.

명시적 대리지표 (추정값)

반감기 · ADMET 점수

서열 기반 랭킹 대리지표이며 임상 PK 값이 아닙니다. in-vitro 혈청 안정성 또는 투과성 분석으로 검증되지 않았습니다. 후보 우선순위 결정용이며 코드베이스에서도 그렇게 레이블링됩니다 (anti-hallucination 면책 조항이 multiobjective.py에 내장).

SST-14 참조 복원 — 파이프라인이 진짜 수용체 아형 분별을 측정함을 확인

네이티브 SST-14는 범 작용제(pan-agonist)로 5종의 소마토스타틴 수용체 아형 모두에 결합합니다 — 즉 SSTR2 선택적이지 않습니다. 플랫폼의 선택성 스크린이 이를 정확히 재현합니다: 변이되지 않은 SST-14 참조는 SSTR2 선택성 마진이 거의 없음을 보여주며, 파이프라인이 실제 수용체 아형 분별을 측정하고 있음을 확인합니다. 후보물질은 변이가 결합을 진짜로 SSTR2 쪽으로 기울였을 때만 양수 선택성 마진을 획득합니다.

결론: 이 플랫폼은 정직한 물리 기반 결합·선택성 수치로 우선순위 결정을 가속화하며, 휴리스틱 추정값을 투명하게 표시합니다. 습식 실험(wet-lab) 결정을 위한 입력으로 설계되었으며, 습식 실험을 대체하지 않습니다.
